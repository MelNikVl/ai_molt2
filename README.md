# Krisha Telegram Bot

Бот мониторит объявления krisha.kz и отправляет персональные уведомления в Telegram.
Построен на aiogram 3.x с 9-шаговым онбордингом, скорингом релевантности и веб-панелью администратора.

---

## Возможности

- **9-шаговый онбординг** (тип → город → район с кнопками → бюджет → комнаты → площадь → заезд → приоритеты → подтверждение)
- **Районы Астаны и Алматы** как inline-кнопки + "Любой" + "Ввести вручную"
- **Фильтрация по району** в URL запроса krisha.kz (`das[district][0]=N`)
- **Приоритеты:** школа рядом, без ремонта, от собственника
- **Дедупликация объявлений** (`core/dedup.py`)
- **Скоринг релевантности** (`core/scorer.py`)
- **Карточки объявлений** с кнопками ❤️ 👎 🔔 📞
- Избранное, блокировка объявлений, отслеживание
- **Подписки по ролям** (1 день / 7 дней / 30 дней)
- **Ежедневная сводка** в выбранный час
- **Парсинг с случайным интервалом 1–5 минут** (защита от блокировки)
- **Веб-панель администратора** на `:8080` с логами, статистикой и ошибками парсера
- **`/settings` с защитой**: «Изменить» или «Оставить как есть»

## Настройка

1. Python 3.11+
2. `pip install -r requirements.txt`
3. Создать `.env` в корне репозитория:

```env
BOT_TOKEN=...
ADMIN_TELEGRAM_ID=...
ADMIN_PASSWORD=change_me
DB_PATH=bot.db
DEAL_TYPE=rent
CITY=astana
MAX_PRICE=200000
MIN_ROOMS=1
MAX_ROOMS=4
BOT_VERSION=2.0.0
TEST=false
```

## Фильтрация по районам krisha.kz

Парсер добавляет `das[district][0]=N` к URL запроса, когда пользователь выбирает район.

**ID районов необходимо проверить в браузере:**
1. Откройте krisha.kz
2. Примените фильтр по нужному району
3. Скопируйте значение `das[district][0]=N` из адресной строки
4. Обновите `ASTANA_DISTRICT_IDS` / `ALMATY_DISTRICT_IDS` в `bot/core/parser.py`

Текущие значения (приблизительные):

| Город | Район | ID |
|---|---|---|

| Астана | Алматы р-н | 14 |

## Бесплатные API карт для поиска по радиусу

Для поиска "в радиусе X км от точки":

- **Telegram `request_location`** — запросить геолокацию прямо в боте (кнопка KeyboardButton с `request_location=True`). Получаете координаты (lat, lon) без внешнего API.
- **Nominatim (OpenStreetMap)** — бесплатный геокодинг, `nominatim.openstreetmap.org`. Лимит: 1 запрос/сек. Позволяет перевести адрес → координаты.
- **2GIS API** — есть бесплатный тариф, лучший охват Казахстана. Геокодинг + поиск по радиусу (`search?point=lon,lat&radius_search=1000`).

Фильтрация объявлений по радиусу на уровне бота: получить координаты пользователя через Telegram, перевести адреса объявлений через Nominatim/2GIS в координаты, считать расстояние формулой haversine.
