# Krisha Telegram Bot (MVP+)

Бот мониторит объявления krisha.kz и отправляет персональные уведомления в Telegram.
Состоит из двух слоёв: рабочий бот (`krisha_bot/`) и новый модернизированный движок (`bot/`).

---

## Возможности (krisha_bot — текущий)

- Онбординг через inline-кнопки (`/start`, `/settings`)
- Защита `/settings`: предлагает "Изменить" или "Оставить как есть"
- Типы сделки: аренда и покупка
- Персональные фильтры: город, тип сделки, цена, метраж
- Парсинг объявлений с случайным интервалом **1–5 минут** (защита от блокировки)
- Подписки по ролям (1 день / 7 дней / 30 дней)
- Ежедневная сводка в выбранный час
- Логирование ошибок парсера в БД
- Веб-панель администратора на `:8080`

## Возможности (bot/ — модернизированный, aiogram 3.x)

- 9-шаговый онбординг (тип → город → **район с кнопками** → бюджет → комнаты → площадь → заезд → приоритеты → подтверждение)
- Районы Астаны и Алматы как inline-кнопки + "Любой" + "Ввести вручную"
- Приоритеты: **ЛРТ** (не метро), школа, ремонт, собственник
- Дедупликация объявлений (`core/dedup.py`)
- Скоринг релевантности (`core/scorer.py`)
- Карточки объявлений с кнопками ❤️ 👎 🔔 📞
- Избранное, блокировка объявлений, отслеживание

## Настройка

1. Python 3.11+
2. `pip install -r requirements.txt`
3. Создать `krisha_bot/.env`:

```env
BOT_TOKEN=...
ADMIN_TELEGRAM_ID=...
ADMIN_PASSWORD=change_me
DB_PATH=krisha_bot/krisha.db
DEAL_TYPE=rent
BOT_VERSION=0.3.0
TEST=false
```

---

## Запуск (текущий бот)

```bash
cd krisha_bot
python main.py
```

Бот пишет логи в `bot.log` в рабочей директории.

## Запуск (модернизированный бот)

```bash
# из корня репозитория
python -m bot.main
```

## Веб-панель (`/admin`)

| Маршрут | Описание |
|---|---|
| `/admin` | Дашборд: пользователи, объявления, запросы, размер БД |
| `/admin/users` | Список пользователей, блокировка, продление подписки |
| `/admin/subscriptions` | Выдача подписки |
| `/admin/logs` | Логи в **реальном времени** (polling каждые 3 сек, последние 20 строк `bot.log`) |
| `/admin/issues` | Ошибки парсера (последние 50, автообновление каждые 30 сек, кнопка очистки) |

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие + онбординг |
| `/settings` | Изменить настройки (с защитой от случайного сброса) |
| `/grant USER_ID ROLE` | Выдать подписку (только для admin) |
